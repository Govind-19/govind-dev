Early on, I just built features end to end... But in every PR review, my lead always forced us to use the same font & colors across the app....

As a beginner i never got why it mattered 🥲... (vibe-coding back then was messy too.... because we have a chat type interface so models were so inconsistent, we had to prompt fonts & colors manually every time 😮‍💨)

I literally never cared about the design 🤷‍♂️....

Now, i notice it everywhere.... the app just feels better to use....

and the moment i see anything related to māhita technologies, i can recognise it instantly.... same font, same colors across every landing page, product, poster designs...

<Pull>Anyone can build a product.... but building a brand means finding that one consistent feel which the user gets used to....</Pull>

thats the major thing i learned from **Jagan Kumar Mudila** 🙏
